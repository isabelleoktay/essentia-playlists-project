# Music Collection Analyzer

## What is it?

This project helps you understand and explore your music collection better. It provides tools to automatically analyze your music and create playlists based on different characteristics.

## How to Use:

1. **Installation**:

   - Clone this repository to your computer.
   - Install required packages using `pip install -r requirements.txt`.

2. **How to generate features**:

   - Make sure your audio data is in the data directory.
   - Run `python audio_analysis_main.py`

3. **How to generate features report**:

   - Run `python music_collection_overview_main.py`

4. **How to generate feature based playlists app**:

   - Run `streamlit run descriptor_queries_app.py`

5. **How to generate similarities based playlists app**:

   - Run `streamlit run similarities_app.py`
